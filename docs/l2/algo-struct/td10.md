### 3)

Pour $n \in \mathbb{N}$ il y a 1 arbre tassé (à gauche) de taille $n$

*hauteur arbre vide =0*

### 4)




Un arbre tassé de hauteur $h$ possède n noeud

$2^{h-1}-1 < n \leq 2^h -1$

($1+2+4+8+16+32+...+2^{h-2} = 2^{h-1}-1$)

- $2^{h-1} -1 < n$

$\iff 2^{h-1} < n+1$

$\iff h-1 < log_2(n+1)$

$\iff h < log_2(n+1) + 1$

- $n \leq 2^h-1$

$\iff n+1 \leq 2^h$

$log_2(n+1) \leq h$

- $log_2(n+1) \leq h < log_2(n+1) + 1$
